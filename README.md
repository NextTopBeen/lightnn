## Modual structure:

### activations

* identity(dense)
* sigmoid
* relu
* softmax
* tanh
* leaky relu
* elu
* selu
* thresholded relu
* softplus
* softsign
* hard sigmoid

### losses

* MeanSquareLoss
* BinaryCategoryLoss
* LogLikelihoodLoss

### initializers

* xavier uniform initializer(glorot uniform initializer)
* default weight initializer
* large weight initializer

### optimizers

* SGD
* Momentum
* RMSProp
* Adam
* Adagrad

### layers

* FullyConnected(Dense)
* Conv2d
* MaxPooling
* AvgPooling
* Softmax
* Dropout
* Flatten
* RNN
